## Basic descriptive statistics

Check this option if you want to see basic descriptive statistics for your data. These include:

---
**For quantitative data**
* Min - minimum value for the variable
* 1st Qu. - first quartile
* Median - median value (50% of the data)
* Mean - mean value
* 3rd Qu. - third quartile
* Max - maximum value for the variable
* NA - number of missing values if present

---
**For qualitative data**

Number of observations in each category and number of NA's (missing values) if present.
